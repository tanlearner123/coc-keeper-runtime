# Formal Adventure Guide

## Adventure

`mad_mansion` / `疯狂之馆`

## Premise

调查员在睡梦中被拖入一间纯白密室，必须在倒计时结束前探索大厅与四个分馆，承担代价，找到离开的方法。

## Recommended Flow

1. `/load_adventure adventure_id:mad_mansion`
2. 每个已加入玩家执行 `/ready`，必要时附带 `character_name`
3. 等 bot 自动发中央大厅开场
4. 让玩家自由决定先进入哪个分馆
5. 用自然消息推进调查，按需要切换 scene/combat
6. 用 `/debug_status campaign_id:<id>` 检查当前房间、线索和鲜血进度

## Scene Nodes

### 1. 中央大厅

- 重点：建立困境、倒计时和四馆选择
- 关键线索：数字钟、强光幻视、四道光幕
- 结果：把玩家送入分馆探索循环

### 2. 贪欲之馆

- 重点：管理员交易、理智代价、危险诱惑
- 关键线索：管理员可被杀、代价永远大于收益
- 结果：玩家获得装备、情报或更深的猜忌

### 3. 生命之馆

- 重点：图书线索、石板交易、感官剥夺
- 关键线索：五个房间的隐喻、生门与死门关系
- 结果：玩家拼出离开条件的一部分

### 4. 死亡之馆

- 重点：尸体、献祭十字架、圣子
- 关键线索：自愿牺牲与死亡结局的区别
- 结果：推进鲜血条件，或制造圣子变量

### 5. 鲜血之馆

- 重点：血池、出口阈值、潜水/开锁思路
- 关键线索：唯有伤痛后的鲜血能打开出去的路
- 结果：满足阈值后开放出口或导向其他结局

## GM Notes

- 这是当前第一个正式结构化模组
- AI 视角是全知 DM，但只能按 state 和 reveal policy 逐步揭露信息
- 当前系统已经把以下内容结构化了：
  - 大厅与四个分馆
  - 倒计时
  - 鲜血出口阈值
  - 关键隐藏状态
  - 多结局入口

## Suggested Tone

- 大厅：冷白、刺眼、像实验场
- 管理员少女：轻佻、恶趣味、故意示弱
- 生命之馆：诡异、学术感、带错位神圣
- 死亡之馆：安静、黑暗、献祭感
- 鲜血之馆：腥臭、机械、临门一脚的压力
