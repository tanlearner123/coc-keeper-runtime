### Handler Parameter Conventions (D-02)

All parameters use `target_` prefix for target stats:

Actor stats: `dex`, `fighting`, `shooting`, `brawl`, `dodge`, `grapple`, `hp`, `hp_max`, `armor`, `damage_bonus`, `weapon_name`, `weapon_type`

Target stats: `target_dex`, `target_fighting`, `target_shooting`, `target_brawl`, `target_dodge`, `target_grapple`, `target_hp`, `target_hp_max`, `target_armor`, `target_damage_bonus`, `target_weapon_name`, `target_weapon_type`

### Combat Handler Signatures

**resolve_fighting_attack(attacker, defender, attacker_roll, defender_roll)** — opposed check
- Critical: attacker_roll == 1
- Fumble: attacker_roll == 100 OR (attacker.fighting >= 50 AND attacker_roll >= 96)

**resolve_shooting_attack(attacker, defender, attacker_roll, range_modifier, recoil_modifier)**
- Critical: attacker_roll == 1
- Fumble: attacker_roll == 100 OR (attacker.shooting >= 50 AND attacker_roll >= 96)

**resolve_brawl_attack(attacker, defender, attacker_roll, defender_roll)** — same as fighting but 1d3+DB damage

**Dodge** — calls resolve_fighting_attack with swapped roles (defender becomes actor)

**resolve_grapple_attack(attacker, defender, attacker_roll, defender_roll)**
- Fumble: attacker_roll == 100 OR (attacker.grapple >= 50 AND attacker_roll >= 96)

### Spell Cast Handler

**_execute_coc_cast_spell** parameters: `spell_key`, `caster_int`, `caster_pow`, `caster_spellcast`, `caster_cthulhu_mythos`, `caster_mp`, `caster_max_mp`, `bonus_dice`, `penalty_dice`
- Critical: rolled == 1
- Fumble: rolled == 100 OR (caster_spellcast >= 50 AND rolled >= 96)
- Spell keys from COC_SPELLS: "contact_ghoul", "energy_bolt", "call_lightning", etc.
